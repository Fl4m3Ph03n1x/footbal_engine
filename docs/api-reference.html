    footbal_engine v1.0.0 API Reference

    Modules

    FootbalEngine

    Database engine. It can read the CSV file and index its contents for faster query searches. It can also query the indexed data.

    FootbalEngine.Populator

    Interface for the populator that fills up the memory table (populates it) with data.

    FootbalEngine.Populator.Cache

    Reads the CSV file, validates and parses its data and then populates the memory tables (the DB) with it's information.

    FootbalEngine.Populator.Parser

    Contains several parsing utility functions to parse the data from the CSV to the Elixir native types.

    FootbalEngine.Populator.Server

    Server for the Cache. Tries to populate it with data and if it gets anything other than a complete success for the indexation, it will keep trying to repopulate the memory tables.

    FootbalEngine.Populator.Storage

    Stores the given data in memory, indexing it as necessary along the way.

    FootbalEngine.Reader

    Client that queries the memory DB for information.
